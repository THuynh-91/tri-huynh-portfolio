## 🚀 **Interactive Terminal**

### How to Access:
1. Look for the **floating terminal button** (💻) in the bottom-right corner
2. Click it to open the interactive command-line interface
3. Type commands and press Enter!

### Available Commands:

| Command | Description |
|---------|-------------|
| `help` | Show all available commands |
| `about` | Learn about Tri |
| `skills` | View technical skills |
| `projects` | List featured projects |
| `contact` | Get contact information |
| `resume` | Download resume (opens in new tab) |
| `joke` | Get a random programming joke 😄 |
| `clear` | Clear terminal screen |
| `exit` | Close terminal |

### Example Usage:
```bash
$ help
Available commands:
  help      - Show this help message
  about     - Learn about Tri
  ...

$ about
👨‍💻 Tri Huynh
🎓 CS Student @ Northeastern University
...

$ joke
Why do programmers prefer dark mode?
Because light attracts bugs! 🐛
```

## 🎮 **Konami Code Easter Egg**

### How to Activate:
Press this sequence on your keyboard:
```
↑ ↑ ↓ ↓ ← → ← → B A
```

### What Happens:
- 🎉 Confetti explosion across the screen!
- 🎮 Special "KONAMI CODE!" message appears
- 🌈 Colorful particles rain down

### Pro Tip:
This is a classic gaming reference. Try it to impress visitors who know their gaming history!

## 📋 **Copy-to-Clipboard Feature**

### Where to Find:

### How it Works:
1. Hover over the email address
2. A copy icon appears
3. Click to copy to clipboard
4. See "✓ Copied!" toast notification

### Benefits:
- No need to manually select and copy
- Instant feedback with animation
- Works on all modern browsers

## 🐛 **Troubleshooting**

### Terminal Not Opening?
- Check browser console for errors
- Try refreshing page
- Make sure JavaScript is enabled

### Konami Code Not Working?
- Make sure page is focused (click anywhere first)
- Type the sequence slowly
- Arrow keys might not work in some browsers

### Copy Not Working?
- Check clipboard permissions
- Try a different browser
- Some browsers block clipboard on HTTP (need HTTPS)
